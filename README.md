# 🚀 ADO AI Dependency Tracker  
**Azure DevOps extension for managing AI/ML dependencies (GPU, compliance, data) in scaled Agile.**  

👉 **Why This Exists**: 70% of AI projects fail due to unmanaged cross-team dependencies.  

## Features  
- Track GPU reservations, compliance approvals.  
- Block deployments until legal tasks are resolved.  

## Setup  
1. Clone this repository.  
2. Install dependencies with `npm install`.
